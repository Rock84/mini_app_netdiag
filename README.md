# ⚡ Micro NetDiag (Agentless RDP Checker)

Ультра-легковесная утилита на базе FastAPI для безагентной диагностики сети между клиентом и Windows Server (VPS). 

Проект создан с упором на минимализм (KISS): минимум JavaScript, никаких сторонних API и нулевая зависимость от внешнего интернета. Идеально для работы в закрытых корпоративных сетях.

## 🚀 Особенности
* **Zero-Touch Client:** Пользователю не нужно ничего скачивать, открывать консоль или делать скриншоты. Достаточно перейти по ссылке.
* **Smart RDP Check:** Скрипт использует Timing Attack в браузере (через `fetch`), чтобы достоверно определить, блокирует ли провайдер клиента порт 3389 (RDP).
* **Server Alibi:** Автоматически замеряет нагрузку на CPU и RAM сервера в момент проверки.
* **Reverse Ping & Tracert:** Сервер автоматически запускает обратную трассировку к IP-адресу клиента через системные утилиты Windows.

## 🛠 Установка и запуск

1. Клонируем репозиторий:
   \`\`\`cmd
   git clone https://github.com/ВАШ_НИК/ИМЯ_РЕПОЗИТОРИЯ.git
   cd ИМЯ_РЕПОЗИТОРИЯ
   \`\`\`

2. Устанавливаем зависимости (всего 3 библиотеки):
   \`\`\`cmd
   pip install -r requirements.txt
   \`\`\`

3. Запускаем сервер:
   \`\`\`cmd
   uvicorn netdiag_micro:app --host 0.0.0.0 --port 8000
   \`\`\`

## 📖 Как использовать
* **Для клиента:** Отправьте ссылку \`http://IP_СЕРВЕРА:8000/client\`
* **Для администратора:** Читайте готовые отчеты по ссылке \`http://IP_СЕРВЕРА:8000/admin\`
